# MemoryOS — Never Forget a Connection

> *Built for the [Bloomathon](https://bloom.diy) hackathon.*

MemoryOS is a voice-first social memory extension that helps you remember every name, detail, and connection you make — turning fleeting whispers into a permanent, searchable vault of social capital.

---

## 🧠 The Problem

Nature set a hard limit at ~150 meaningful relationships (Dunbar's Number). In a world of global networks, most people lose 90% of a meeting's value within 48 hours — forgetting niche interests, small promises, and the sparks of alignment that turn a contact into a partner.

**You aren't losing names. You're losing leverage.**

---

## ✨ Features

| Feature | Description |
|---|---|
| **Neural Ingress** | Save a connection detail in under 10 seconds with voice input |
| **Cognitive Retrieval** | Search your network by vibe, location, or semantic tag |
| **Encrypted Synapse** | End-to-end encryption — your social graph belongs to you |
| **Network Pulse** | Dashboard showing network composition, gaps, and relationship decay |
| **Sunday Quiz** | Weekly active-recall quizzes based on your new connections |

## 🚀 Getting Started

MemoryOS is a static web landing page. No build step is required.

1. **Clone the repository**
   ```bash
   git clone https://github.com/ramnnn2006/bloomathon.git
   cd bloomathon
   ```

2. **Open locally**
   ```bash
   # Using any static file server, e.g.:
   npx serve .
   # or simply open index.html in your browser
   open index.html
   ```
